# MonkeyBrush++

# Introduction

## Dependencies
 - GLFW
 - GLEW

## Building
MonkeyBrush++ is based on Cmake.

```sh
$ git clone https://github.com/MonkeyBrushDev/MonkeyBrushCpp.git
$ cd MonkeyBrushCpp
$ mkdir build && cd build
$ cmake .. -DCMAKE_BUILD_TYPE=Release
$ make
```
